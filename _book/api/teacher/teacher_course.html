课程管理

课程模块(course)

添加课程

路由：

PUT /teacher/course

参数：

参数 描述 类型 必填 描述
courseNumber 课程号 string true
courseIndex 课序号 string true
academy 开课学院 string true
teacherNumber 任课教师工号 array true 一门课可能有多个教师上课
startTime 课程起始时间 string true YYYY-MM-DD
endTime 课程结束时间 string true YYYY-MM-DD

响应

{

}

路由

GET /teacher/course

路由
