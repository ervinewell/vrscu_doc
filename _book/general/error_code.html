错误码

错误码

为了在请求出错时更准确地定位错误，除了HTTP请求原生的状态码之外，我们还约定了以下错误码：

错误码 描述
-1 未知错误
1001 请求参数非法
2001 用户已存在
2002 用户名或密码错误
2003 用户不存在
2003 注册角色受限
3001 API权限不足
4001 资源不存在
4002 资源禁止访问
4003 资源重复
