# 响应

## 格式

所有请求响应会以以下格式给出：

```
{    
    code: 200, // 状态码
    msg: 'success', // 提示信息 
    data: {} // 数据
}
```

## 状态码

下述表格规定了状态码的**一般含义**，更多信息由`msg` 字段给出。

| 状态码 | 描述 |
| :--- | :--- |
| 200/2XX | 请求成功 |
| 400 | 请求参数有误 |
| 401 | 授权信息（用户名/密码/token）有误 |
| 403 | 资源禁止访问 |
| 404 | 资源未找到 |
| 413 | 请求提交数据太大 |
| 500 | 其他错误 |
